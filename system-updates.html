  • Version 2.1.1 2018-10-10
    sha256 update-2.1.1.img: 915f336b314dbe73e41c0b843f4ddf689a0db842028217c0f76bc7ad8aa3f72d

    Changes

    • SAFETY FEATURES
      • Improved detection and handling of communication errors.
    • NEW FEATURES
      • Overall performance improvements, e.g. handling of complex and long App sequences in one task.
      • Improved Desk design with ergonomic aspects and enhanced handling functionalities.
      • New functionalities can be added via Apps and services.
      • Better control of the “monitored stop”.
      • User interface improvements and bug fixes.

    Notes for Panda research

    • If your system is older than 1.3, version 1.3.2 (see below) needs to be installed first.
    • With this update the functionality of the External Activation Device will change (for detailed information please click here), beyond that the original USER MANUAL that you received with purchasing the Panda robot does still apply.

    • Do NOT perform a factory reset

      This will uninstall the Franka Control Interface – FCI and your Research App Package that you have to request again from support@franka.de

    • Taught positions could be slightly off due to controller improvements.
  • Version 1.3.2 2018-06-27
    sha256 update-1.3.2.img: 52d86b84bb640a29c87703517f7220f7ddfc2400f6e34c6ed310a4b5313cc36b

    • Update FCI interface, requires libfranka >= 0.4.0 and franka_ros >= 0.5.0
      • Important: Please read the new entries of the libfranka documentation about libfranka and signal processing / rate limiters.
      • FCI filters are deactivated by default.
      • Robot limits are updated.
      • Added first and second derivatives of last commanded signals.
      • Fix issue with max path pose deviation with external torque controller.

    • Improved joint and Cartesian impedance controller.
    • Improved self collision avoidance algorithm.
    • Improved overall stability and bug fixes.
